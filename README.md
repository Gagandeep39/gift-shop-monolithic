## Payment

- Razor Pay
  - Card - `4111 1111 1111 1111`
  - CVV - Any 3 digit number
  - Date - Any Future Date
- Stripe
  - Card - `4242 4242 4242 4242`
  - CVV - Any 3 digit number
  - Date - Any Future Date
